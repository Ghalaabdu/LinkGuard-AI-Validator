 LinkGuard AI
A Modern URL Validator with Heuristic AI Analysis.

Features
AI Risk Scoring: Analyzes URLs for phishing patterns.

Bi-lingual: Supports Arabic and English (RTL/LTR).

Modern UI: Built with Tailwind CSS and Glassmorphism.

 How the AI Works
The tool uses Heuristic Analysis to evaluate risks based on:

Protocol security (HTTPS vs HTTP).

Suspicious keywords detection.

URL length and masking patterns.
